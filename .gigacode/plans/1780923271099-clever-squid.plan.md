# План доработки: Модальное окно редактирования связей модуля

## Фон

В проекте no-code-ide есть холст с блоками (источники, модули, приёмники). На данный момент при двойном клике на блоке модуля ничего не происходит. Требуется реализовать модальное окно для редактирования связей модуля с источниками и приёмниками данных, аналогичное окну AddModulePage.

## Текущая структура

### Ключевые файлы:
- **Editor.tsx** - компонент холста с графиком
- **ModuleBlock.ts** - класс отрисовки блока модуля
- **ModuleProperties.tsx** - отображение свойств модуля (текущее)
- **AddModulePage.tsx** - пример модального окна для добавления модуля
- **AddSourcePage.tsx** - модальное окно для добавления источника
- **AddDestinationPage.tsx** - модальное окно для добавления приёмника
- **ElementPanel.tsx** - панель элементов с двойным кликом
- **ProjectContext.tsx** - контекст состояния проекта
- **SelectionContext.tsx** - контекст выбранного элемента

### Структура данных:
```typescript
interface Module {
    name: string;
    location: string;
    sources: DataSource[];    // Связанные источники
    destinations: DataDestination[];  // Связанные приёмники
}

interface DataSource {
    name: string;
    ipAddress: string;
    tcpPort: number;
}

interface DataDestination {
    name: string;
    url: string;
    dependency: ExternalDependency;
}
```

## Реализация

### 1. Создать модальное окно ModuleConnectionsModal
**Файл**: `src/react/main/components/ModuleConnectionsModal.tsx`

Компонент должен:
- Открываться с переданным модулем и списком всех доступных источников/приёмников
- Позволять выбирать источники и приёмники для связи с модулем
- Иметь две колонки: слева - доступные источники, справа - доступные приёмники
- Кнопки добавления/удаления связей
- Обрабатывать подтверждение и отмену

### 2. Обновить ModuleBlock для обработки двойного клика
**Файл**: `src/react/main/components/ModuleBlock.ts`

Добавить обработчик события двойного клика, который будет отправлять кастомное событие с ID блока.

### 3. Обновить Editor для обработки двойного клика на модуле
**Файл**: `src/react/main/components/Editor.tsx`

Добавить обработку события двойного клика на блоке, который:
- Проверяет тип блока (должен быть 'module')
- Открывает модальное окно ModuleConnectionsModal
- Передаёт текущий модуль и список всех источников/приёмников проекта

### 4. Обновить ModuleProperties для отображения связей (опционально)
**Файл**: `src/react/main/components/ModuleProperties.tsx`

Можно добавить отображение количества связанных источников и приёмников.

### 5. Интеграция в ProjectEditor
**Файл**: `src/react/main/containers/EditorContainer.tsx`

Убедиться, что Editor обернут в необходимые провайдеры контекста.

## Планируемое поведение

**Реализовано:**

При двойном клике на блоке модуля:
1. ✅ Открывается модальное окно `ModuleConnectionsModal`
2. ✅ Показываются два списка: доступные источники и доступные приёмники (используется Select с multiple)
3. ✅ Пользователь может выбрать источники и приёмники для связи с модулем
