# План реализации: Генератор Java-кода для Spring Boot-приложения

## Контекст

Проект no-code-ide - это визуальный редактор схем с использованием библиотеки `@gravity-ui/graph`. В проекте есть модули, источники данных и приёмники. Есть существующая функция `buildProject`, которая скачивает Spring Boot шаблон и выполняет сборку.

## Требования

1. **Генерация Java-кода**: Генерировать Java-код на основе данных проекта (project.json, schema, файлы модулей)
2. **Сохранение в папку generated**: Код должен сохраняться в `/generated/имя_сгенерированного_приложения`
3. **Базовый пакет**: Все классы в одном пакете `com.example.имя_проекта` без подпакетов
4. **ID-based имена**: Использовать ID для имен классов (Module1, Source1, Destination1)
5. **Классы конфигурации**: Генерировать сервисы модулей с @Component
6. **Клиенты приёмников**: Генерировать клиенты для приёмников с @Component
7. **Заглушки**: Использовать System.out.println вместо реальных вызовов команд
8. **Типы приёмников**: Поддерживать PostgreSQL и Kafka с разной конфигурацией
9. **UI для настройки**: Добавить UI для настройки конечных точек приёмников (PostgreSQL: имя БД, схема, таблица, столбец; Kafka: topic)
10. **Сохранение настроек**: Сохранять настройки в схеме проекта и отображать метки на соединениях

## Анализ текущей структуры

### Типы данных (`src/types/model.d.ts`)

```typescript
interface Project {
    name: string;
    location: string;
    modules: Module[];
    sources: DataSource[];
    destinations: DataDestination[];
    dependencies: ExternalDependency[];
    schema: Schema;
}

interface Module {
    name: string;
    location: string;
    sources: DataSource[];
    destinations: DataDestination[];
}

interface DataSource {
    name: string;
    ipAddress: string;
    tcpPort: number;
    command?: DataSourceCommand;
    commandParams?: Record<string, string | number | boolean>;
}

interface DataDestination {
    name: string;
    url: string;
    targetType?: string;
    databaseName?: string;
    schemaName?: string;
    tableName?: string;
    columnName?: string;
    topic?: string;
    dependency: ExternalDependency;
}

interface DataSourceCommand {
    name: string;
    description: string;
    parameters: DataSourceCommandParameter[];
}
```

### Структура связей (из schema.edges)

Связи между источниками и модулями определяются по `sourceBlockId` и `targetBlockId` в `schema.edges`.

### Существующая функция `buildProject`

Функция уже реализована в `src/electron/utils/project.ts` и:
1. Создаёт папку `generated`
2. Скачивает Spring Boot шаблон с start.spring.io
3. Распаковывает шаблон
4. Выполняет Gradle сборку

Нужно расширить эту функцию для генерации Java-кода.

## Описание генерируемого кода

### Класс приёмника (PostgreSQL)

```java
package com.example.project;

import org.springframework.stereotype.Component;

@Component
public class Destination1Client {
    private static final String DATABASE_NAME = "mydb";
    private static final String SCHEMA_NAME = "public";
    private static final String TABLE_NAME = "users";
    private static final String COLUMN_NAME = "data";
    private static final String URL = "jdbc:postgresql://localhost:5432/mydb";

    public void sendData(String data) {
        System.out.println("Отправка данных в Destination1 (POSTGRESQL)");
        System.out.println("Данные: " + data);
    }

    public void sendBatchData(java.util.List<String> dataList) {
        System.out.println("Отправка пакета данных в PostgreSQL БД: " + DATABASE_NAME + ", схема: " + SCHEMA_NAME + ", таблица: " + TABLE_NAME);
        dataList.forEach(data -> System.out.println("Данные в столбце " + COLUMN_NAME + ": " + data));
    }
}
```

### Класс приёмника (Kafka)

```java
package com.example.project;

import org.springframework.stereotype.Component;

@Component
public class Destination1Client {
    private static final String TOPIC = "my-topic";
    private static final String BOOTSTRAP_SERVERS = "localhost:9092";

    public void sendMessage(String message) {
        System.out.println("Отправка сообщения в Kafka topic: " + TOPIC + " на сервер: " + BOOTSTRAP_SERVERS);
        System.out.println("Сообщение: " + message);
    }

    public void sendBatchMessages(java.util.List<String> messages) {
        System.out.println("Отправка пакета сообщений в Kafka topic: " + TOPIC);
        messages.forEach(msg -> System.out.println("Сообщение: " + msg));
    }
}
```

### Класс сервиса модуля

```java
package com.example.project;

import org.springframework.stereotype.Component;

@Component
public class Module1Service {
    private final Destination1Client destination1Client;

    public Module1Service(Destination1Client destination1Client) {
        System.out.println("Инициализация модуля Module1");
        this.destination1Client = destination1Client;
    }

    public void processDataFromSource1() {
        System.out.println("NEVA_GET_STATUS - обработка данных от источника Source1");
    }

    public void sendDataToDestination1() {
        System.out.println("Отправка данных в приёмник Destination1 (POSTGRESQL)");
        destination1Client.sendData("Данные из модуля Module1");
    }
}
```
